# E-commerce Data Analysis & Dashboard

## 📌 Overview

This project analyzes e-commerce sales data to uncover trends, revenue distribution, and customer behavior insights. It combines Python-based data analysis with an interactive Power BI dashboard.

---

## 🧰 Tools Used

* Python (Pandas, Matplotlib)
* Power BI
* VS Code

---

## 📊 Features

* Data cleaning and preprocessing
* Revenue analysis by country
* Monthly sales trend analysis
* Top-selling products identification
* Interactive Power BI dashboard with filters

---

## 📈 Key Insights

* United Kingdom contributes the majority of revenue
* Sales show strong growth in later months of the year
* A small number of products generate a large portion of revenue
* Customer purchases are concentrated in specific regions

---

## 📁 Project Structure

project1/
│
├── data/                # Raw dataset
├── output/              # Cleaned dataset
├── scripts/             # Python code

└── README.md

---

## 🚀 How to Run

1. Run Python script to clean data:
   python scripts/main.py

2. Open Power BI file:
   powerbi/ecommerce_dashboard.pbix
